# Neatfinity - Professional Cleaning Services Website

A modern, beautiful Next.js website for Neatfinity cleaning business.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
